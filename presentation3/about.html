ABOUT BUGATTI





ბუგატის დაარსება

ავტომობილები ეტორე ბუგატი იყო ფრანგული მაღალი წარმადობის ავტომობილების მწარმოებელი. კომპანია დაარსდა 1909 წელს მაშინდელ გერმანულ ქალაქ მოლშეიმში, ალზასში, იტალიაში დაბადებული ინდუსტრიული დიზაინერის, ეტორე ბუგატის მიერ. მანქანები ცნობილი იყო სილამაზისა და მრავალი რბოლის გამარჯვებებით. ცნობილი ბუგატის ავტომობილებში შედის Type 35, Type 41 "Royale", Type 57 "Atlantic" და Type 55 სპორტული მანქანა.

ბუგატის მიღწევები

Bugatti-ს მანქანები ძალიან წარმატებული იყო ავტორბოლებში. Type 10-მა პირველივე რბოლაში ოთხი პოზიცია დაიკავა, ხოლო 1924 წლის Type 35 გახდა ერთ-ერთი ყველაზე წარმატებული სარბოლო მანქანა. Bugatti-მ გაიმარჯვა ტარგა ფლორიოში 1925-1929 წლებში. ყველაზე დასამახსოვრებელი გამარჯვება იყო 1939 წლის ლე მანში.
